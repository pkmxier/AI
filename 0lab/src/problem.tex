\CWHeader{Лабораторная работа \textnumero 0}

\CWProblem{
Требуется сформировать/получить два набора данных соответствующие следующим критериям:
\begin{enumerate}
    \item Один из датасетов должен представлять собой корпус документов. Язык, источник и тематика произвольна
    \item Второй датасет должен содержать категориальные, количественные признаки. Для данного датасета определить предсказываемые признаки (для задачи регрессии и классификации). Если такого признака нет, спроектировать
\end{enumerate}

Данные датасеты будут в дальнейшем использованы в оставшихся лабораторных работах.

По каждому датасету построить распределения признаков (в случае корпуса документов – построить распределение слов) и объяснить имеющуюся картину. Вычислить статистические характеристики признаков. Обнаружить и решить возможные проблемы с данными. Если решить данную проблему невозможно, объяснить почему
}

{\bfseries Требования:}
\begin{enumerate}
    \item Датасеты должны быть уникальны
    \item Исходный код должен быть написан в одном код стайле
    \item Должен быть указан источник данных
\end{enumerate}

{\bfseries Выбранные датасеты:}
\begin{enumerate}
    \item Корпус документов - статьи BBC по 5 жанрам: business, entertainment, politics, sport, tech
    
    http://mlg.ucd.ie/datasets/bbc.html
    \item Данные котировок акций американских компаний(NYSE, NASDAQ)
    
    https://www.kaggle.com/borismarjanovic/price-volume-data-for-all-us-stocks-etfs
\end{enumerate}
